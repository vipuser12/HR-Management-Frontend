# 🏥 HR Management System (Full-Stack)

A modern, full-stack Human Resources Management platform designed to streamline employee data, manage payroll, and handle administrative tasks with high security and efficiency.

---

## 🚀 Live Demo
* **Frontend (Vercel):** [https://hr-management-frontend-six.vercel.app/](https://hr-management-frontend-six.vercel.app/) 
* **Backend API (MonsterASP):** [https://hrm-api-v2.runasp.net/swagger](https://hrm-api-v2.runasp.net/swagger)

---

## 🛠️ Tech Stack

### **Frontend:**
* **React.js (Vite)** - For a fast, component-based user interface.
* **Tailwind CSS** - For modern, responsive, and clean styling.
* **React Router Dom** - Handling client-side routing and navigation.
* **Axios** - Efficient management of API requests.

### **Backend:**
* **ASP.NET Core Web API 8.0** - Scalable server-side architecture.
* **Entity Framework Core** - ORM for seamless database interaction.
* **SQL Server** - Reliable relational database.
* **JWT (JSON Web Token)** - Secure, token-based authentication.
* **Swagger UI** - Comprehensive API documentation and testing.

---

## ✨ Key Features
- [x] **Secure Authentication:** Role-based access control using JWT tokens.
- [x] **Employee Management:** Full CRUD (Create, Read, Update, Delete) functionality for employee records.
- [x] **Payroll Management:** Efficient salary tracking, updates, and payroll processing.
- [x] **Dynamic Dashboard:** Real-time data visualization and company statistics.
- [x] **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
- [x] **CORS Enabled:** Secure cross-origin communication between Vercel and MonsterASP.
